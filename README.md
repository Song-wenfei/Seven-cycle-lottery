# Seven Cycle Lottery

Seven Cycle Lottery is a lightweight decision-making game designed to help users choose an activity or event for the day in a randomized manner.

#### How It Works

1. Create a list of up to seven events, activities, or options.
2. Entries may be repeated within the list to increase their probability of being selected.
3. Once per day, the game randomly selects one event from the list, which becomes the recommended activity for that day.

#### Features

* **Customizable Event List:** Add, edit, or remove events at any time.
* **Weighted Selection:** Repeating an event increases its likelihood of being chosen.
* **Daily Draw:** One randomized selection is available each day.
* **Manual Override:** Users may change the selected event if desired.
* **Cycle Reset:** The seven-day cycle can be reset at any time, allowing a fresh start with the same or a new set of events.

#### Purpose

The game provides a simple and engaging way to introduce variety into daily routines, support decision-making, and encourage exploration of different activities.


## Run locally

From this folder:

```powershell
python -m http.server 4177
```

Then open `http://localhost:4177`.

## Install on iPhone

Host the folder on any static web host, open the URL in Safari on the iPhone, tap Share, then choose Add to Home Screen. After the first load, the app works offline from the Home Screen.
